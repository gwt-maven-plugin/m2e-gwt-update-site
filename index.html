m2e-gwt-update-site by gwt-maven-plugin

Update site for m2e-gwt project

This repository hosts P2 update site for the m2e-gwt project. Two repositories will be available: a working (snapshots) one under the gwt-maven-plugin.github.com/m2e-gwt-update-site/nightly URL. a stable (releases) one under the gwt-maven-plugin.github.com/m2e-gwt-update-site/releases URL.

(codehaus-mojo-gwt-maven-plugin-users@googlegroups.com)

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/gwt-maven-plugin/m2e-gwt-update-site

get the source code on GitHub : gwt-maven-plugin/m2e-gwt-update-site
